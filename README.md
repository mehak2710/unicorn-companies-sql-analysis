# Unicorn Companies Business Analytics Using SQL

## Project Overview

This project analyzes a global Unicorn Companies dataset using **MySQL** to uncover valuable business insights about startup valuations, industries, countries, investors, and funding stages.

The project demonstrates practical SQL skills used in Business Intelligence (BI), Data Analytics, Venture Capital research, and startup ecosystem analysis. It follows a structured workflow from data exploration and validation to business analysis using SQL queries.

---

## Project Objectives

- Analyze the global distribution of unicorn companies.
- Identify the countries and cities producing the most unicorn startups.
- Discover the most popular industries among unicorn companies.
- Analyze company valuations and investor participation.
- Explore funding stages and portfolio exits.
- Generate business insights using SQL aggregation and analytical queries.

## Business Questions Answered

This project answers the following business questions through SQL analysis:

1. How many unicorn companies are present in the dataset?
2. How many countries and industries are represented in the global unicorn ecosystem?
3. Which countries have the highest number of unicorn companies?
4. Which countries have more than 20 unicorn companies?
5. Which cities are home to the largest number of unicorn companies?
6. Which industries have the highest concentration of unicorn companies?
7. Which industries have the highest average number of investors?
8. Which unicorn companies have the highest valuations?
9. What is the average valuation of unicorn companies?
10. Which countries have the highest average unicorn valuation?
11. Which unicorn companies were founded after 2015?
12. How has the number of unicorn companies changed by founding year?
13. Which unicorn companies have attracted the highest number of investors?
14. What is the average number of investors per unicorn company?
15. How are unicorn companies distributed across different financial stages?
16. Which companies have the highest number of deal terms?
17. Which companies have the highest number of portfolio exits?
18. How can unicorn companies be categorized into **Mega**, **Large**, and **Emerging Unicorns** based on their valuation?
19. What are the overall business metrics of the unicorn ecosystem, including total companies, countries, industries, average valuation, and investor statistics?
